**What Actually Happens Every Month**

Every month, the vault runs through a simple rhythm:

  ------- ----------------------------------------------------------------
  **1**   The vault runs. Income is earned from four sources.

  **2**   Expenses are paid. Gas fees, transaction costs --- covered
          automatically.

  **3**   If there is surplus: the floor increases slightly.

  **4**   If there is no surplus: the floor pauses. Nothing collapses.

  **5**   You see your updated balance. You decide what to do with it.
  ------- ----------------------------------------------------------------

No drama. Just cycles.

**What is \'The Floor\'?**

This is the most important concept. Pay attention.

+-----------------------------------------------------------------------+
| **The Floor Rule**                                                    |
|                                                                       |
| The floor is the minimum price at which you can redeem your LOOP.     |
|                                                                       |
| → It can stay the same.                                               |
|                                                                       |
| → It can increase.                                                    |
|                                                                       |
| → It can NEVER go down.                                               |
|                                                                       |
| Not in a bad month. Not in a market crash. Not by anyone\'s decision. |
|                                                                       |
| Enforced at the smart contract level. Not a promise. A rule in code.  |
+-----------------------------------------------------------------------+

When the vault earns surplus, a tiny amount is added to the floor. Not a
lot. Not a promise of big returns. Just a small, real, permanent
increase. Like adding one brick at a time.

When there\'s no surplus, the floor pauses. No bricks added. But none
removed either.

***The floor never retreats. That\'s not a marketing line. It\'s a math
rule.***
